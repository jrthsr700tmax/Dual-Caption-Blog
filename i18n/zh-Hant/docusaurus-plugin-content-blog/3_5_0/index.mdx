# YouTube™ 雙字幕 3.5.0 更新說明

### 自定義字體樣式

點擊擴展圖標，彈出設置頁面，支持黑暗模式。

自定義選項：字體，字號，顏色，背景顏色，間距，以及字幕順序。

<img src={light} alt="light.png" width="340" height="472" />
<img src={dark} alt="dark.png" width="340" height="472" />

<Tip>懸停在圖標上將顯示設置代表含義。</Tip>

<img src={tip} alt="tip.gif" width="340" height="472" />

### 單字幕樣式

切換到單字幕模式後，可以單獨設置單字幕樣式。

### 其他優化

移除中文繁體轉簡體程序庫，減少程序體積，提升性能，因為 YouTube 官方已經修復了這個錯誤。

<Tip>有時候會出現字幕丟失的情況，這主要是因為YouTube翻譯出錯。</Tip>
